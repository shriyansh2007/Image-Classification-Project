# Image Classification using PyTorch

## Overview

This project implements an Image Classification Model using Convolutional Neural Networks (CNNs) on the CIFAR-10 dataset.

## Technologies Used

- Python
- PyTorch
- NumPy
- Matplotlib
- Scikit-Learn
- Google Colab

## Features

- Data preprocessing
- Image augmentation
- CNN architecture
- Accuracy evaluation
- Precision, Recall and F1-score
- Confusion Matrix
- Loss Curve Visualization

## Dataset

CIFAR-10 Dataset

Classes:
- Airplane
- Automobile
- Bird
- Cat
- Deer
- Dog
- Frog
- Horse
- Ship
- Truck

## Results

- Training Accuracy: 85%+
- Validation Accuracy: 75-80%
